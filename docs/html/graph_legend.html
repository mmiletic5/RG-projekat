Ова страница објашњава како тумачити графове који су направљени doxygen-ом.

Размотримо следећи пример:

/*! Невидљива класа због одсецања */
class Invisible { };
/*! Одсечена класа, веза наслеђивања је скривена */
class Truncated : public Invisible { };
/* Класа која није документована doxygen коментарима */
class Undocumented { };
/*! Класа која је наслеђена јавним наслеђивањем */
class PublicBase : public Truncated { };
/*! Шаблонска класа */
template<class T> class Templ { };
/*! Класа која је наслеђена заштићеним наслеђивањем */
class ProtectedBase { };
/*! Класа која је наслеђена јавним наслеђивањем */
class PrivateBase { };
/*! Класа коју користи наслеђена класа */
class Used { };
/*! Надкласа која наслеђује неки број других класа */
class Inherited : public PublicBase,
protected ProtectedBase,
private PrivateBase,
public Undocumented,
public Templ<int>
{
private:
Used *m_usedClass;
};

Правоугаоници имају следеће значење:

  • Пуни сиви правоугаоник представља структуру или класу за коју је граф направљен.
  • Правоугаоник са црним оквиром означава документовану структуру или класу.
  • Правоугаоник са сивим оквиром означава недокументовану структуру или класу.
  • Правоугаоник са црвеним оквиром означава документовану структуру или класу заза коју нису све релације наслеђивања/садржавања приказане. Граф је одсечен ако излази из специфицираних оквира.

Стрелице имају следећа значења:

  • Тамноплава стрелица се користи да прикаже релацију јавног извођења између двеју класа.
  • Тамнозелена стрелица се користи за заштићено наслеђивање.
  • Тамноцрвена стрелица се користи за приватно наслеђивање.
  • Љубичаста испрекидана стрелица се користи ако класа садржи или користи друга класа. Стрелица је означена променљивом/променљивама кроз које је показивана класа или структура доступна.
  • Жута испрекидана стрелица означава везу између примерка шаблона и и шаблонске класе из које је инстанцирана. Стрелица је означена параметрима примерка шаблона.
